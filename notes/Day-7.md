## Functions in Python
- Functions are used to perform a specific task repeatedly
- Defined using the `def` keyword
- Function names should be unique
- Functions must be defined before calling
- Functions improve code reusability and readability

## Function Arguments Practiced
- Positional arguments
- Default arguments
- Keyword arguments
- Arbitrary arguments using *args
- Arbitrary keyword arguments using **kwargs

## Return Statement
- Functions can return values using `return`
- Returned values can be stored in variables
- `return` exits the function with a value

## Key Notes
- Local variables exist only inside the function
- Built-in functions do not need to be defined
- Indentation is very important in functions

## Lists in Python
- Lists are ordered and mutable collections
- Lists allow duplicate values
- Lists can store multiple data types
- Indexing starts from 0
- Negative indexing is supported
- Slicing can be used to access ranges of elements

## List Operations Practiced
- Indexing and slicing
- Membership checking using `in`
- List comprehension
- append()
- sort()
- reverse()
- index()
- count()
- insert()
- extend()
- List concatenation
